Ultra Name Lab!

A highly customizable, open-source fantasy name generator. Ultra Name Lab lets you mix and match different data sets of names to generate something totally unique

How does it work?

Like pretty most fantasy name generator, names are generated using Markov models. This means it takes a large data set of real names and builds a statistical model of the language (or at least, the letter sequences in a language). It then uses that model to generate new names.

Usage tips

Adding an English data set at a low weight to exotic-sounding languages will produce names that sound a little more familiar

Adding multiple similar data sets with a generic, well-rounded feel. e.g. French, Spanish and Italian will produce generic European names
