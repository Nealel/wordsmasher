wordSMASHER!

A highly customizable, open-source fantasy name generator. WordSMASHER lets you mix and match different data sets of names to generate something totally unique

How does it work?

Like a lot of fantasy name generator, names are generated using Markov models. This means it takes a large data set of real names and builds a statistical model of the language (specifically, the probability of the next letter, given the previous let of letter). It then that model to generate new names.

Usage tips

Adding a data set for your native language (e.g. English for English-speakers) at a low weight to exotic-sounding languages will produce names that sound a little more familiar

Adding multiple similar data sets with a generic, well-rounded feel. e.g. French, Spanish and Italian will produce generic European names

Adding a 'generic' data set will include all the more specific sets that fall under it. e.g. European includes French, and French includes French (female)
